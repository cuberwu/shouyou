# 首右辅助码文档总览

欢迎来到文档中心。左侧是按主题组织的完整目录，右侧这篇总览用于帮助你快速判断“先看什么、再练什么”。

## 推荐学习路径

1. **导读：先完成环境搭建**
   - 先看《从零开始搭建首右辅助码学习环境》
   - 目标：把输入法、码表和方案切到可练习状态

2. **双拼：先把高频规则跑通**
   - 再看《双拼键位速记：声母、韵母与零声母规则》
   - 目标：让声母 + 韵母组合输入稳定下来

3. **双形：建立字根与拆分习惯**
   - 然后看《字根分布与拆分方向：先记结构再记编码》
   - 目标：形成一致的拆分思路，减少反复试错

4. **应用：配置效率功能**
   - 接着看《简码与符号输入：效率跃升的关键设置》
   - 目标：把常用输入压缩为更少按键

5. **排错：建立问题闭环**
   - 最后看《常见问题排查：重码、词库与快捷键冲突》
   - 目标：遇到问题时能快速定位并恢复稳定输入

## 每日练习建议（15 分钟）

- **5 分钟**：复习当天对应文档的 1-2 条核心规则
- **8 分钟**：做高频字词跟打，观察重码与选词节奏
- **2 分钟**：记录今天遇到的 1 个问题，次日优先排查

## 如何使用这个文档中心

- 左侧目录用于**按主题跳转**与快速定位
- 进入具体文章后，可使用页面内目录定位小节
- 建议按“导读 → 双拼 → 双形 → 应用 → 排错”顺序学习

持续按路径练习一周，通常就能形成稳定的输入节奏与纠错能力。
